# war3icon

## 特点
在线2.1W个魔兽世界图标合集，可以方便转换边框方案，点击就能自动生成对应的亮，暗，被动图标的BLP，也能批量导入png图标

## 使用教程
右上角既可打开，搜索功能之支持英文的，目前所有图标没有好好分列，有空再说！
